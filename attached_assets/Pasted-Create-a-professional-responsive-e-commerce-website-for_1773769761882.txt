Project Structure:

Create a main folder for the project.

Inside the project, create an images folder. Within images, create subfolders for each brand like Samsung, Oppo, Xiaomi, etc. Place all product images in their respective brand folders. Images should be named including the model, e.g., galaxys25fe.jpg.

Include viewdetails.html for product detail pages. Include a clean layout with:

Large product image on one side

Product name, price, specifications table, and a WhatsApp "Buy Now" button on the other side

The design should not include any placeholder text like “Contact for price”, only the real price should be displayed

Include main.js in src to handle loading product details dynamically based on URL parameters.

Use HTML for structure, CSS for styling, and JavaScript for dynamic behavior.

The project should not require any backend initially. It should be fully front-end with static files and JavaScript to render products and their details.
